# 서버 증설 횟수

## 링크
- [서버 증설 횟수](https://school.programmers.co.kr/learn/courses/30/lessons/389479)

```java
import java.util.*;

class Solution {
    public int solution(int[] players, int m, int k) {
        int answer = 0;
        int n = 0;
        
        Queue<Integer> queue = new LinkedList<>();
        
        for (int i = 0; i < players.length; i++) {
            while (!queue.isEmpty() && (i == queue.peek())) {
                queue.poll();
                n--;
            }
            
            int toAdd = (players[i] / m) - n;
            
            if (toAdd > 0) {
                for (int j = 0; j < toAdd; j++) {
                    queue.add(i + k);
                }
                answer += toAdd;
                n += toAdd;
            }
        }
        
        return answer;
    }
}
```
## 풀이
- queue에서 가져온 서버의 만료 시각이 현재 시각과 같으면 반납합니다.
- 현재 시각에 필요한 서버 수는 `players[i] / m` 입니다.
- 이미 `n` 대가 운영 중이므로 추가로 필요한 서버 수는 `tdAdd = (players[i] / m) - n` 입니다.
- `tdAdd > 0` 이면 서버를 `toAdd`대 증설하고, 각 서버의 만료 시각인 `i + k` 를 queue에 추가합니다.
- `toAdd <= 0` 이면 서버를 증설하지 않는다.
